Web (Next.js 14 App Router)

Environment
- Copy `.env.example` to `.env` and set `NEXT_PUBLIC_API_BASE` to your server URL (e.g., http://localhost:4000).

Scripts
- dev: `npm run dev` (http://localhost:3000)
- build: `npm run build`
- start: `npm start`

Routes
- /login, /signup, / (feed), /new, /u/[username], /p/[id]
